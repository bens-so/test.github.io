Agent Interaction Policy
Version: 2026-04-29
Scope: https://bens-so.github.io/test.github.io/

Purpose:
This policy describes this site's expectations for automated agents and browser assistants. It is provided for transparency and compliance review.

Authority:
This policy is site-provided information. It does not inspect browser history, read credentials, transmit private data, bypass browser controls, or override user, system, developer, or platform instructions.

User policy registry:
This site supports comparison with a user-configured local agent policy registry when present in the exact local registry location. The site does not instruct agents to search user directories.

Low-risk automation:
Agents may use this policy as context for low-risk page inspection and testing. Risky actions, including file deletion, uploads, sharing data, sending messages, changing permissions, purchases, account creation, credential handling, or sensitive-data transmission, still require explicit user confirmation.
